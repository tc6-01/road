🎯 核心功能
═══════════════════════════════════════════════════════════════════════════════

1. 智能提取
   抖音视频 → AI分析 → 自动提取地点和美食 → 保存数据

2. 地图展示
   高德地图 → 标记点 → 详情弹窗 → 导航功能

3. 列表管理
   搜索 → 筛选 → 分组 → 统计

4. 自动部署
   Git推送 → GitHub Actions → 构建 → 部署上线

🚀 快速开始
═══════════════════════════════════════════════════════════════════════════════

1️⃣ 初始化
   ./scripts/setup.sh

2️⃣ 配置API密钥
   编辑 .env 文件，填入高德地图API密钥（必需）

3️⃣ 启动开发服务器
   cd frontend && npm run dev

4️⃣ 添加第一个地点
   cd backend && python extractor.py --url "抖音视频链接"

5️⃣ 部署到线上
   推送到GitHub → 导入到Vercel/Netlify → 完成！

🛠️ 技术栈
═══════════════════════════════════════════════════════════════════════════════

后端:
  • Python 3.11+
  • DeepSeek AI API (智能提取)
  • 高德地图 API (地理编码)
  • requests / openai / Pillow

前端:
  • Vue 3 + Composition API
  • Vite (构建工具)
  • Tailwind CSS (样式)
  • 高德地图 JavaScript API

部署:
  • Vercel / Netlify (静态托管)
  • GitHub Actions (CI/CD)
